📱 METHOD 1: DRAG & DROP (EASIEST!)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Run app in simulator:
   $ npm run ios

2. Drag ANY .apns file onto the simulator window

3. 🎉 Notification appears!


💻 METHOD 2: COMMAND LINE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
$ cd test-notifications
$ xcrun simctl push booted org.reactjs.native.example.FadeBook appointment-booked.apns


🧪 METHOD 3: TEST ALL AT ONCE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
$ cd test-notifications
$ chmod +x test-all-notifications.sh
$ ./test-all-notifications.sh

⚠️  TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
No notification appearing?
→ Check: Settings → Notifications → FadeBook → Allow Notifications (ON)
→ Reset: xcrun simctl privacy booted reset notifications org.reactjs.native.example.FadeBook

Simulator not running?
→ Start: npm run ios

JSON error?
→ Validate at: https://jsonlint.com
