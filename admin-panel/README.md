# Barber Admin Panel

React web application for managing barber shops, barbers, services, and bookings.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:3001`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Features

- Dashboard with statistics and recent activity
- Manage barbers
- Manage shops
- Manage services and pricing
- View and manage bookings
- Analytics dashboard
- System settings

## Tech Stack

- React 18
- TypeScript
- Vite
- React Router
- Zustand (State Management)
- TanStack Query
- Lucide React (Icons)
